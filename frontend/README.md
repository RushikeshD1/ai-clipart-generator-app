## Overview
**AI ClipArt Generator** is a mobile app built with **Expo React Native** that converts normal images into AI-generated clipart styles using a **Cloudflare image-to-image model** through a **Node.js backend deployed on Vercel**. Users can select styles, preview AI images, save them to a gallery, and share them via multiple channels like WhatsApp or system share.

---

## Features

- **AI Image Conversion**
  - Cartoon
  - Flat Illustration
  - Anime
  - Pixel Art
  - Sketch / Outline

- **Loading States**
  - Skeleton loaders while AI images are generated for smooth UX.

- **Gallery Management**
  - Save AI-generated images to a personal gallery.
  - Download or share images via multiple channels.

- **Style Selector Tab**
  - Choose the desired style before generating images.

- **Image-to-Style Workflow**
  - Select an image from your mobile.
  - Generate AI style image with preview.
  - Save or share.

- **Gallery Tab**
  - View all saved images.
  - Download images or share via WhatsApp/other apps.
